🏏 KKR IPL Performance Dashboard (2008–2023)

📊 Overview

This project is a Power BI dashboard analyzing Kolkata Knight Riders (KKR) performance from 2008 to 2023.

🔥 Key Insights

- Win rate: ~51%
- Best season: 2014 (~77%)
- Strong death-over batting performance
- Weakness in death-over bowling
- Best opponent: Punjab Kings
- Struggles against: Mumbai Indians

📌 Features

- Batting & Bowling Analysis
- Phase-wise Performance (Powerplay, Middle, Death)
- Player Performance Insights
- Venue & Toss Impact Analysis
- Interactive Visualizations

🛠 Tools Used

- Power BI
- DAX
- Data Visualization


🚀 How to Use

1. Download the ".pbix" file
2. Open in Power BI Desktop
3. Explore interactive visuals
